# Contributing to navi-SAD

Thank you for your interest in contributing. See the full contributing guide on the [docs site](https://project-navi.github.io/navi-SAD/how-to/contributing/).

## Quick start

```bash
git clone https://github.com/Project-Navi/navi-SAD.git
cd navi-SAD
uv sync --extra dev
uv run pre-commit install
make all
```

## Open problems

The project maintains a list of [open problems](https://project-navi.github.io/navi-SAD/research/open-problems/) at various difficulty levels.
